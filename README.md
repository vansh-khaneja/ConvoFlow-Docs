# ConvoFlow Documentation

Comprehensive documentation for ConvoFlow - a powerful no-code platform for building custom chatbots with visual workflows.

## Getting Started

### Development

```bash
# Install dependencies
npm install

# Run development server
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view the documentation.

### Building for Production

```bash
# Build the documentation site
npm run build

# Start production server
npm start
```

## Documentation Structure

The documentation is organized into the following sections:

- **Getting Started**: Introduction, Installation, Architecture, Quick Start
- **Customization**: Adding Nodes, Creating Tools, Node Configuration, Styling
- **Guides**: Building Workflows, API Integration, Deployment, Troubleshooting
- **Reference**: Node Types, Tool Reference, API Reference
- **Examples**: Basic Chatbot, Knowledge Base Bot, Multi-Step Workflows

## Features

- Clean, modern design inspired by Medusa documentation
- Responsive sidebar navigation
- Code examples with syntax highlighting
- Step-by-step guides
- Comprehensive API and reference documentation

## Contributing

To add or update documentation:

1. Create or edit pages in the `app/` directory
2. Update the sidebar navigation in `components/Sidebar.tsx` if needed
3. Follow the existing documentation style and format
4. Test locally with `npm run dev`

## Tech Stack

- **Next.js 16** - React framework with App Router
- **TypeScript** - Type safety
- **Tailwind CSS** - Styling
- **React** - UI components
